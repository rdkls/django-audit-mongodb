.. topic:: Overview

	django-audit adds auditing of any models you create in
	`django <http://djangoproject.com>`_ recording any changes in a
	`MongoDB <http://www.mongodb.org/>`_ collection. This way you'll always have
	a history all changes that have been made to your data.
